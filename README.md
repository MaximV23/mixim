Adopted from the [MIXIM](https://gitlab.esat.kuleuven.be/Iness.BenGuirat/mixim) codebase ([paper](https://cosicdatabase.esat.kuleuven.be/backend/publications/files/conferencepaper/3403)), as a teaching tool for the Advanced Privacy Technologies course.

# Installation Instructions 
```bash
cd mixim
sudo apt install python3
python3 -m venv venv
source venv/bin/activate
pip install numpy, pandas, simpy and matplotlib
python3 main.py
python3 run_test.py
```
